TM Single Category Products

This module allows you to display 4 main modules (Bestsellers, Featured, Specials, Latest) with the products from certain category in tab view.

After the module is installed you can configure it on the module settings page.

Settings:
  1. Module Name
    the name of the module;
  2. Category
    allows to select the existing product category;
  3. Tabs
    enables/disables tab view. If disabled, there is an option to select module view type;
  4. Product Type
    allows to select module type. Available when the Tabs option is disabled.
    The following values are available:
    • Specials;
    • Bestsellers;
    • Featured;
    • Latest.
  5. Specials
    enables/disables Specials module;
  6. Bestsellers
    enables/disables Bestsellers module;
  7. Featured
    enables/disables Featured module;
  8. Latest
    enables/disables Latest module;
  9. Featured Products
    allows to select product list for the module Featured. Available when Featured module is enabled or Product type option is set to Featured;
  10. Limit
    the number of products to display;
  11. Width
    product image width;
  12. Height
    product image height;
  13. Status
    module status (enabled/disabled).
