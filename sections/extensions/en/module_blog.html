Blog

The following extension allows you to manage posts and news connected to your store.

Main features:

  • creating articles;
  • creating categories;
  • creating authors;
  • commenting articles and replying to the comments;
  • view report;
  • displaying products and articles related to the certain article;

In order to change Blog global settings navigate to System->Settings. Select the store where you want to change the settings out of the list and click Edit.

Blog settings can be seen in Blog section:
  1. Status
    blog status (enabled\disabled);
  2. Limit
    number of posts per page;
  3. Description Symbol Limit
    number of symbols in the article description on the main Blog page (not on single article!);
  4. Comment Auto approval
    enables/disables automatic comment approval by administrator;
  5. Author Information
    enables/disables information about the author on article page;
  6. Related Article
    enables/disables related articles;
  7. Share with social site
    enables/disables social share buttons;
In the side panel you can see new tab - Simple Blog. It contains the following options:
  1. Article
    list of created articles;
  2. Category
    list of created categories;
  3. Author
    list of created authors;
  4. Article
    Comment - list of added comments;
  5. Report
    views report.

TM Blog Articles

This module allows to display blog articles by categories.

After the module is installed you can configure it on the module settings page.

Settings:
  1. Status
    module status (enabled\disabled).
  2. Module Name
    the name of the module;
  3. Article limit
    number of articles to display;
  4. Readmore Button Show
    enables/disables Read more button;
  5. Date Show
    enables/disables date;
  6. Author Show
    enables/disables article author;
  7. Comments Show
    enables/disables the number of comments per article;
  8. Image Show
    enables/disables article image;
  9. Image Width
    image width;
  10. Image Height
    image height;
  11. Description Symbol Limit
    number of symbols in article description;
  12. Category
    you can set the category to pull articles from. There is also a possibility to display most popular articles or recently added articles.

TM Blog Category

This module allows you to display blog articles according to different options.

After the module is installed you can configure it on the module settings page.

Settings:
  1. Status
    module status (enabled\disabled).
  2. Module Name
    name of the module;
  3. Search Article
    enables/disables search.
