Introdução O que você vai encontrar nesse pacote e como usá-lo.

Obrigado por adquirir um template de OpenCart. Esse manual consiste em diversas partes que demonstram todo o processo de instalar e gerenciar seu site OpenCart do início. Nós nos esforçamos para que este manual fosse o mais claro e fácil de acompanhar possível.

O que é OpenCart?

OpenCart é uma plataforma de comércio eletrônico grátis e de código aberto. OpenCart oferece uma base confiável e profissional para construir uma loja online de sucesso. Essa base apetece a uma variedade de usuários, desde desenvolvedores web procurando uma interface amigável até lojistas querendo lançar seu negócio na web pela primeira vez. OpenCart tem uma série de dispositivos que te dão controle completo sobre a personalização da sua loja. Com as ferramentas do OpenCart, você pode fazer sua loja atingir seu ápice. Saiba mais

A estrutura de arquivos

Os arquivos a seguir estão incluídos no seu template:

  • documentation - contém um manual detalhado sobre a instalação do seu template;
    • documentation.html - o arquivo de documentação principal. Você está aqui :)
  • screenshots - contém imagens do tema funcionando;
  • sources - contém um arquivo fonte do Adobe Photoshop;
  • themeXXX - contém:
    • themeXXX(full).zip - o arquivo com a engine do OpenCart, o template e os dados de teste (para uma instalação do zero);
    • themeXXX(update).zip - esse arquivo tem o template sem os dados de teste (para instalar por cima de uma loja existente);
    • themeXXX(extensions).zip - esse arquivo com arquivos XML;
    • unzip.php - o arquivo usado para extrair arquivos das versões comprimidas (full ou update) quando você subir seus arquivos para o servidor.
  • info.txt - contém informações sobre o arquivo;
  • fonts_info.txt - contém informações sobre as fontes usadas no template.

Preparação O que você precisa pra fazer funcionar

Para rodar a loja do OpenCart, você precisa se certificar de algumas coisas.

Software de Edição

Para se sentir confortável em trabalhar com um tema do OpenCart, recomendamos que você baixe os programas necessários.
Os programas necessários podem variar entre templates, então aqui vamos cobrir o que você precisa em geral:

  1. Você pode precisar do Adobe Photoshop instalado. Você pode usá-lo para editar os arquivos .PSD e vai precisar dele para editar gráficos e imagens do template.
  2. Para editar o código fonte do template, você vai precisar de um editor de código como o Adobe Dreamweaver, o Notepad++, o Sublime Text, etc.
  3. Para subir os arquivos para seu servidor de hospedagem, você vai precisar de um gerenciador de FTP, como o FileZilla, CuteFTP, etc.
  4. Por favor, verifique os seguintes tutoriais sobre como extrar arquivos do WinZip (no PC) e do StuffitExpander (no MAC).

Hospedagem

Já que o OpenCart é uma aplicação baseada em PHP/MySQL, você precisa preparar o seu ambiente para rodá-lo. Se você já tem um serviço de hospedagem, certifique-se de que os requerimentos de sistema e a configuração do servidor estão prontas para rodar sites OpenCart.

Você pode rodar o OpenCart no seu computador, usando um servidor local. Para criar um servidor local, use programas de localhost como o WAMP, AppServ, MAMP, etc. Todos esses podem ser instalados facilmente e usados para rodar o OpenCart.

Os seguintes tutoriais podem ajudá-lo a configurar um ambiente de desenvolvimento local:

  • Como instalar o ambiente de desenvolvimento AppServ
  • Como instalar o ambiente de desenvolvimento WAMP
  • Como instalar o ambiente de desenvolvimento MAMP

Começando O que clicar para fazer a coisa funcionar

Instalação completa

Essa opção é usada quando você não tem nenhum sistema OpenCart instalado no seu servidor. O arquivo 'themeXXX(full).zip' usado nesse guia, contém o motor do OpenCart e o tema com dados de teste, assim você não precisa fazer upload do OpenCart em si separadamente. Os dados de teste vão fazer seu site OpenCart ficar igual a página de preview do seu template, com todos os produtos e módulos de teste para o seu site.

 

Atualização

Use essa opção se você já tem um site OpenCart e só quer mudar sua aparência instalando um tema novo.

 

Instalação completa

O método de instalação completa deve ser usado se você quiser começar um site OpenCart do zero. Nós assumimos que você já passou pelas etapas de preparação e já tem tudo o que é necessário.

Criando um Banco de Dados

Depois que você subir os arquivos para o seu servidor, pode preparar um novo banco de dados para o seu site OpenCart. Você pode criar um banco usando a ferramenta de criação de banco de dados do seu painel de controle (normalmente é o PhpMyAdmin).

Com o PhpMyAdmin você pode criar um novo banco de dados em 3 passos simples (observe os slides abaixo):

  1. Clique no botão de Banco de Dados no menu do topo para ver a listagem de bancos de dados.
  2. Escreva o nome do banco de dados no campo "Create new database" (Criar novo banco de dados).
  3. Clique no botão de criar à esquerda.

Agora você pode ver a listagem do seu banco de dados, clicando no nome dele para obter acesso ao banco.

 

  • 1. Clique no botão "Database" para acessar a lista de bancos

  • 2. Coloque um nome no campo "Database".
    3. Clique em "Create" (Criar).

  • 4. Agora você pode ver o seu banco de dados na listagem. Clique no nome dele para acessá-lo.

 

Você também pode dar uma olhada no tutorial em vídeo, que detalha Como criar um banco de dados.

Se tiver qualquer problema na criação do banco, entre em contato com seu serviço de hospedagem para obter ajuda e assistência.

Subir e descompactar

Para começar a fazer o OpenCart funcionar, você precisa subir os arquivos pro servidor. Você pode fazer isso usando um Gerenciador de Arquivos do seu serviço de hospedagem ou através de um programa de FTP.

  1. Abra a pasta do template.
  2. Selecione os arquivos 'unzip.php' e o 'themeXXX(full).zip' e suba ambos para o seu servidor ( Como subir arquivos para o servidor, Como subir arquivos para o servidor usando o cPanel (WebHost Manager, WHM)).
  3. Escreva o caminho para o arquivo 'unzip.php' no seu server (http://seu_dominio/unzip.php) no seu navegador. Você verá a seguinte tela:
  4. Você verá a seguinte tela:

Imagem 1. Tela inicial do Unzip.php

  1. Na caixa de seleção 'Choose your zip file', selecione o arquivo 'themeXXX(full).zip'.
  2. No campo 'Unzip to', especifique o diretório onde você deseja extrair os arquivos.
  3. Clique no botão 'Unzip' para prosseguir.
Certifique-se de que você aplicou as permissões corretas para o diretório onde você deseja extrair os arquivos. Permissões devem ser CHMOD 755 ou 777 dependendo da configuração do seu servidor.

Instalação do OpenCart

Abra seu navegador e digite o caminho para o diretório do OpenCart na barra de endereços (exemplo: http://seu_dominio/opencart). Você deve ver a seguinte tela de instalação:

 

Instalação passo 1 Licença

 

Imagem 2. Instalação do OpenCart Passo 1. Licença

Leia a licença do OpenCart e, se você concordar com os termos, dê um check em I agree to the license e clique no botão Continue.

 

Passo 2 Pré-Instalação

 

Imagem 3. Instalação do OpenCart. Passo 2. Pré-Instalação

Nesse passo, o motor do OpenCart verifica o seu servidor de hospedagem: se ele está dentro dos requerimentos mínimos, e se as permissões do diretório estão corretas.

Certifique-se de que você aplicou as permissões corretas nas pastas do OpenCart, como descrito abaixo:

  • chmod 0755 ou 0777 image/
  • chmod 0755 ou 0777 image/cache/
  • chmod 0755 ou 0777 image/data/
  • chmod 0755 ou 0777 system/cache/
  • chmod 0755 ou 0777 system/logs/
  • chmod 0755 ou 0777 download/
  • chmod 0755 ou 0777 config.php
  • chmod 0755 ou 0777 admin/config.php.

 

Passo 3 Configuração

 

Imagem 4. Instalação do OpenCart. Passo 3. Configuração

Agora, você precisa colocar os detalhes da conexão com seu banco da dados. Lembra que você criou o banco de dados MySQL na preparação? Agora você deve inserir:

  • o nome do servidor do banco de dados
  • o nome do banco de dados
  • o nome do usuário do banco de dados
  • a senha do usuário do banco.

Se tiver dificuldades em conseguir esses detalhes, entre em contato com seu serviço de hospedagem.

Você também precisa criar uma conta de administrador da loja. Use um nome de usuário, senha e email que você consiga lembrar. Esses dados serão usados para entrar no painel administrativo do OpenCart, que serve para gerenciar sua loja.

 

Passo 4 Pronto!

 

Imagem 5. Instalação do OpenCart. Passo 4. Pronto!

Esse é o último passo na instalação do OpenCart. Você tem duas opções: visitar o front-end do OpenCart, seu site de e-commerce; ou o back-end, que é o painel administrativo.

Você pode dar uma olhada nesse tutorial de instalação: OpenCart. How to install OpenCart engine and a template (using “fullpackage”) .

Certifique-se de deletar o diretório install do seu servidor. Você não poderá entrar na sua loja enquanto ele existir.

Os seguintes arquivos precisam ter sua permissão setada para 644 ou 444, para que ninguém mais mexa neles:

  • config.php
  • index.php
  • admin/config.php
  • admin/index.php
  • system/startup.php.

A instalação está completa. Entre no front-end da loja, escrevendo seu_dominio na barra de endereços do seu navegador. Para acessar o painel administrativo, escreva seu_dominio/admin.

Atualização

Por favor, verifique se sua versão do OpenCart é igual ou superior a exigida pelo template. Todas as versões do OpenCart estão disponíveis no site oficial do OpenCart.

  1. Para instalar o template do OpenCart você precisa subir seus arquivos pro servidor. No pacote do template de OpenCart que você adquiriu, encontre o arquivo "themeXXX(update).zip" e suba-o para a pasta raíz da sua instalação do OpenCart.
    Se quiser, dê uma olhada no detalhado tutorial em vídeo sobre como subir os arquivos para o servidor de hospedagem.
    Os arquivos do template não vão afetar sua loja de forma alguma.
  2. Quando terminar, suba o arquivo unzip.php do pacote do template para o servidor, no mesmo diretório do arquivo anterior.
  3. Assim que subir esse arquivo, abra seu site no seu navegador e coloque o nome do arquivo - unzip.php - ao final do endereço. Deve ficar assim: seu_site.com/unzip.php.
  4. Na caixa "Choose your zip file", selecione themeXXX(update).zip e clique em "Unzip". Isso vai iniciar a desompressão dos arquivos no seu servidor para o mesmo diretório. Isso pode demorar um pouco.
  5. Digite seu_site.com/install/ na barra de endereço do seu navegador.
  6. Depois disso, siga as instruções de instalação que aparecerem na sua tela.
  7. Entre no seu painel administrativo, navegue para Extensões (Extensions) -> Instalador de extensão (Extension Installer) e envie os arquivos de ThemeXXX(extensions). Então, navegue para Extensões -> Modificações (Modifications) e clique no botão de atualizar (Refresh) no topo à direita.

Você pode dar uma olhada no tutorial de instalação: OpenCart. How to install a template over the existing store (using “update” package).

Configurando Extensões

Chat Ao Vivo Olark

Para colocar o chat ao vivo como você viu na demonstração, você vai precisar se registrar no site oficial da Olark. Depois de inserir os dados no serviço deles, você vai receber o código que você deve inserir em catalog\view\theme\themeXXX\template\common\footer.tpl antes da tag </body>.

Política de Cookies

Para seguir as condição da Lei de Cookies você precisar adicionar módulos apropriados para o seu site. O módulo vai mostrar um aviso sobre cookies usados no seu site. Você pode usar o seguinte texto:

Validação

Ambos marcação do HTML e estilos CSS usados em nossos temas são semânticamente corretos e válidos. Entretanto, alguns erros no validador do W3C ainda podem ocorrer. Criar código 100% válido pelas diretrizes do W3C elimina o uso de tecnologias web modernas, como o CSS3 e marcação HTML5. Nossa proposta é prover experiências ricas ao usuário com temas de alta qualidade e, para isso, temos que quebrar algumas regras.
