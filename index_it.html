Introduzione Cosa trovi nel pacchetto e come utilizzarlo

Grazie per aver acquistato un template OpenCart. Questo manuale è suddiviso in diverse sezioni per illustrare l'intera procedura di creazione e e gestione di un sito OpenCart, a partire da zero. Abbiamo fatto del nostro meglio per realizzare un manuale che sia il più possibile chiaro e facile da seguire.

Cos'è OpenCart?

OpenCart è una piattaforma e-commerce gratuita e open source per rivenditori online. OpenCart è in grado di offrire una base professionale ed affidabile per realizzare uno store online di successo. Il prodotto è pensato per diverse tipologie di utenti, dall'esperto sviluppatore web in cerca di un'interfaccia leggera, al proprietario di attività commerciale che si appresta a introdurre per la prima volta la sua attività sul web. OpenCart include un gran numero di funzioni che ti daranno un pieno controllo sul sito, con la possibilità di personalizzare al massimo il tuo negozio online. Con gli strumenti di OpenCart potrai esprimere tutto il potenziale della tua attività online. Altre Informazioni

Struttura File

Il pacchetto del template include i seguenti elementi:

  • documentation - contiene un manuale completo sul template e sull'installazione del motore;
    • documentation.html - Il file principale per la documentazione. E' il file su cui ti trovi in questo momento :)
  • screenshots - contiene alcune schermate del tema;
  • sources - contiene il file source di Adobe Photoshop;
  • themeXXX - contiene:
    • themeXXX(full).zip - l'archivio che contiene il motore OpenCart, il template stesso e i dati campione del template (per l'installazione clean);
    • themeXXX(update).zip - l'archivio che contiene il template senza dati campione (per installare su store preesistente);
    • unzip.php - il file utilizzato per estrarre i file dall'archivio .zip del pacchetto full/update quando caricato sul server host.
  • info.txt - contiene informazioni sull'archivio;
  • fonts_info.txt - contiene informazioni sui font utilizzati nel template.

Prepara Cosa ti serve per far funzionare il tutto

Prima di eseguire uno store OpenCart, saranno necessari alcuni preparativi.

Software per l'Editing

Per lavorare comodamente con un tema OpenCart, ti consigliamo di scaricare tutte le applicazioni necessarie. Puoi trovare la lista del software necessario sulla pagina di anteprima del template.
I requisiti possono essere diversi a seconda del template, per cui quella che segue è solo una lista di base dei requisiti generali:

  1. Potresti aver bisogno di Adobe Photoshop, che potrai utilizzare per modificare i file source .PSD. Sicuramente ne avrai bisogno se desideri modificare il design del template, la grafica e le immagini.
  2. Per modificare il codice sorgente, servirà un editor come Adobe Dreamweaver, Notepad++, Sublime Text, ecc.
  3. Per caricare i file sul server, potresti avere bisogno di un gestore FTP Manager, come FileZilla, CuteFTP, ecc.
  4. Leggi i tutorial che seguono per una guida su come estrarre i file dagli archivi utilizzando WinZip(PC) e StuffitExpander(MAC).

Hosting

Dato che OpenCart è un'applicazione basata su PHP/MySQL, sarà necessario preparare un ambiente hosting per eseguire OpenCart.

Se hai un hosting live, verifica che questo risponda ai Requisiti di Sistema e che sia pronto all'utilizzo di siti web OpenCart.

E' anche possibile eseguire OpenCart localmente sul tuo computer utilizzando il server locale. Per creare un server di hosting locale, utilizzare applicazioni localhost come WAMP, AppServ, MAMP, ecc. Queste applicazioni possono essere installate facilmente come qualunque programma ed essere utilizzato per eseguire OpenCart.

Guarda i tutorial che seguono su come configurare un ambiente di sviluppo locale:

  • Come installare l'ambiente di sviluppo web AppServ
  • Come installare l'ambiente di sviluppo web AppServ WAM
  • Come installare l'ambiente di sviluppo web AppServ XAMP.

Installazione Completa

Questa modalità di installazione va selezionata se non hai nessuna installazione OpenCart sul tuo server. L'archivio 'themeXXX(full).zip' utilizzato in questa guida contiene il motore OpenCart e il tema con dati campione, così non sarà necessario caricare il motore OpenCart separatamente. I dati campione daranno al vostro sito OpenCart lo stesso aspetto della pagina di anteprima, con tutti gli esempi di prodotti, i moduli e il testing.

 

Installazione Aggiornamento (Update)

Selezionare questa opzione se già hai un sito OpenCart e vuoi solo cambiarne l'aspetto installando un nuovo tema.

 

Installazione Completa

L'installazione completa va selezionata se si desidera creare un sito OpenCart a partire da zero. Partiamo dal presupposto che abbiate già completato tutta la fase di preparazione e che abbiate già tutto il necessario.

Creare un Database

Quando avete finito di caricare i file sul server potete preparare un nuovo database dal vostro sito OpenCart. Potete creare il database utilizzando lo strumento di gestione database che trovate sul vostro pannello di controllo per hosting (solitamente PhpMyAdmin).

Con lo strumento phpMyAdmin sarete in grado di creare un nuovo database in 3 semplici passi (vedete le immagini qui sotto):

  1. Cliccare sul pulsante Database nella barra menù in alto per accedere alla lista database.
  2. Inserire il nome del database nel campo Crea nuovo database ("Create new database").
  3. Cliccare sul pulsante Create, a destra.

Dovreste ora essere in grado di vedere il vostro database nell'elenco database. Cliccare sul suo nome per accedervi.

 

  • 1. Cliccare sul pulsante Database per accedere alla lista database..

  • 2. Inserire il nome del database.
    3. Cliccare sul pulsante Create.

  • 4. Dovreste ora essere in grado di vedere il vostro database nell'elenco database. Cliccare sul suo nome per accedervi.

 

Potete anche vedere un video completo su Come creare un database.

Se avete problemi a creare il database, chiedete assistenza al vostro provider.

Caricare ed Estrarre

Per iniziare a lavorare con OpenCart, devi caricare i file sul tuo server. Puoi farlo utilizzando il gestore file del tuo host, oppure utilizzando un'applicazione esterna per gestione FTP.

  1. Apri la directory del pacchetto template.
  2. Seleziona 'unzip.php' e 'themeXXX(full).zip' e caricali sul server( Come caricare i file sul server, Come caricare i file sul server usando cPanel (WebHost Manager, WHM)).
  3. Digita il percorso del file 'unzip.php' sul tuo server (http://tuo_dominio/unzip.php) nel tuo browser.
  4. Ora dovresti vedere la seguente schermata:

Imm 1. Schermata iniziale Unzip.php.

  1. Nella casella 'Choose your zip file', seleziona il file caricato, 'themeXXX(full).zip'.
  2. Nel campo 'Unzip to', specifica la directory in cui vuoi estrarre i file.
  3. Clicca su 'Unzip' per proseguire.
Non dimenticare di impostare tutti i permessi necessari per la directory in cui vuoi estrarre i file. I permessi dovrebbero essere CHMOD 755 o 777, a seconda delle configurazioni del tuo server.

Installazione OpenCart

Apri il tuo browser e digita il percorso della tua directory OpenCart nella barra degli indirizzi (es. http://tuo_dominio/opencart). A questo punto dovresti vedere la schermata iniziale dell'installazione di OpenCart:

 

Installazione Fase 1 Licenza

 

Imm. 1. Installazione OpenCart. Fase 1. Licenza

Leggere i termini di licenza OpenCart e, se li accetti, spuntare la casella I agree to the license e cliccare sul pulsante Continue.

 

Fase 2 Pre-Installazione

 

Imm. 3. Installazione OpenCart. Fase 2. Pre-Installazione

In questa fase, il motore OpenCart effettuerà dei controlli sul tuo server: verificherà che le impostazioni rispondano ai requisiti di OpenCart e che i permessi delle directory siano impostati correttamente.

Assicuratevi di aver impostato i permessi corretti per il motore OpenCart, come descritto di seguito:

  • chmod 0755 o 0777 image/
  • chmod 0755 o 0777 image/cache/
  • chmod 0755 o 0777 image/data/
  • chmod 0755 o 0777 system/cache/
  • chmod 0755 o 0777 system/logs/
  • chmod 0755 o 0777 download/
  • chmod 0755 o 0777 config.php
  • chmod 0755 o 0777 admin/config.php.

 

Fase 3 Configurazione

 

Imm. 4. Installazione OpenCart. Fase 3. Configurazione

In questa fase dovrai inserire le informazioni di connessione al database. Come ricorderai, nella fase di preparazione avevi creato un database MySQL database. Ora devi indicare

  • nome server del database
  • nome del database
  • nome utente database
  • password utente database.

Se hai difficoltà a trovare queste informazioni, chiedi assistenza al tuo provider.

Dovrai anche creare un account amministratore store. Inserire il nome utente dell'Administrator, la password e l'E-mail. Queste informazioni saranno utilizzate per accedere al pannello di amministrazione OpenCart per gestire il tuo store.

 

Fase 4 Finito!

 

Imm 5. Installazione OpenCart. Fase 4. Finito!

Questa è l'ultima fase dell'installazione del motore OpenCart. Ora hai due opzioni: visitare il front end OpenCart, il tuo sito e-commerce, oppure il back end, cioè il pannello di amministrazione.

Qui puoi vedere un tutorial per l'installazione: OpenCart. Come installare il motore OpenCart e un template (usando “fullpackage”) .

Non dimenticare di eliminare la directory "install" dal tuo server. Fino a quando la cartella sarà presente sul tuo server non sarai in grado di accedere al tuo store.

I seguenti file devono essere impostati su 644 o 444 per evitare che qualcuno possa modificarli:

  • config.php
  • index.php
  • admin/config.php
  • admin/index.php
  • system/startup.php.

Assicurati che le cartella vqmod e le cartelle vqmod/vqcache siano scrivibili (755 o 777).

L'installazione è completa. Per accedere al front end dello store, digita tuo_dominio nella barra degli indirizzi del tuo server. Per accedere al pannello di amministrazione OpenCart, digita tuo_dominio/admin.

Installazione Aggiornamento (Update)

Assicurati che la versione del tuo motore OpenCart risponda ai requisiti del template. Tutte le versioni di OpenCart sono disponibili sul Sito Ufficiale OpenCart.

  1. Per installare un template OpenCart devi caricarne i file sul tuo server. Nel pacchetto template di OpenCart, individua l'archivio "themeXXX(update).zip" e caricalo sul root dell'installazione OpenCart.
    Puoi anche vedere il video tutorial su come caricare i file sul server.
    I file del template non avranno nessun impatto negativo sul tuo store.
  2. Compiuta questa operazione, caricare sul server il file unzip.php dal pacchetto, nella stessa cartella dell'archivio precedente.
  3. Appena finito, apri il tuo sito con un browser e aggiungi il nome file - unzip.php alla URL. Dovrebbe essere così: iltuosito.com/unzip.php.
  4. Nel menù a tendina "Choose your zip file", seleziona themeXXX(update).zip e clicca "Unzip." Inizierà ad estrarre i file sul tuo server nella stessa directory. Ci vorrà un po' di tempo.
  5. Digita iltuosito.com/install/ nella barra degli indirizzi del tuo browser.
  6. Poi segui le istruzioni di installazione che appariranno sullo schermo.

Qui puoi trovare un tutorial per l'installazione: OpenCart. Come installare un template su di uno store preesistente (usando il pacchetto “update”).

Benvenuto su vQmod

"vQmod™" (cioè "Virtual Quick Mod") è un sistema di override creato per evitare di dover modificare i file core. Invece di applicare le modifiche direttamente sui file core, le modifiche vengono create come file script xml di tipo cerca/sostituisci. Questi file script vengono analizzati durante il caricamento della pagina con ogni file core "sorgente" che viene caricato con la funzione php "include" o "require". La sorgente viene poi aggiornata con le modifiche sul file script e salvata in un file temporaneo. Quel file temporaneo va poi a sostituire il file originale durante l'esecuzione. Il file sorgente originale quindi non viene mai alterato. Questo crea dunque delle modifiche "virtuali" al core durante l'esecuzione senza che i file core vengano effettivamente modificati.

Se non hai vQmod installato sul tuo sito (http://code.google.com/p/vqmod/), devi installarlo.

Come installare vqmod usando Autoinstaller

  1. Scarica l'ultima versione che abbia "opencart" nel titolo, su http://code.google.com/p/vqmod/downloads/list
  2. Usando l'FTP, carica la cartella "vqmod" dallo zip alla root del tuo store opencart.
  3. Assicurati che la cartella vqmod e le cartelle vqmod/vqcache siano scrivibili (755 o 777).
  4. Assicurati anche che index.php e admin/index.php siano scrivibili.
  5. Se non sei sicuro di quale sia quella corretta, prova prima 755.
  6. Se poi ricevi un messaggio d'errore sui permessi, allora prova con 777.
  7. Vai su http://www.iltuosito.com/vqmod/install.
  8. Dovresti vedere un messaggio di conferma di installazione riuscita. In caso contrario, controlla i permessi indicati qui su e riprova.
  9. Carica la homepage del tuo store e verifica che funzioni.
  10. Usando l'FTP, verifica che ci siano nuovi file "vq" nella cartella "vqmod/vqcache".
  11. In caso affermativo, sei pronto a iniziare a scaricare o creare script vQmod.
