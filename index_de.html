Einführung Was eine Vorlage enthält und wofür man es nutzen kann

Vielen Dank, dass Sie OpenCart Vorlage gekauft haben. Die jeweilige Dokumentation besteht aus mehreren Teilen und beschreibt den ganzen Installationsprozess und alle Einstellungen der OpenCart Website bei Null. Wir haben alles Mögliche gemacht, um diese Anleitung bequem und einfach für Sie zu machen.

Was ist eigentlich OpenCart?

OpenCart ist eine kostenlose E-Commerce-Plattform mit offenliegendem Quellcode. OpenCart ermöglicht es professionelle und sichere Basis zu erstellen, um einen erlogreichen Online-Shop aufzubauen. Zur Opencart Gemeinschaft gehören mehrere Benutzer wie: erfahrene Web-Entwickler, die bequemes Interface suchen, Inhaber eines Online-Shops, die zum ersten Mal eigenes Online Business starten. OpenCart stellt Ihnen eine Menge Funktionen zur Verfügung, damit Sie Ihren Online-Shop völlig verwalten können. Dank der Werkzeuge, die Ihnen OpenCart anbietet, ist es möglich solchen Online-Shop zu erstellen, der allen Ihren Zwecken entsprechen wird. Mehr

Dateistruktur

Die Vorlage besteht aus den folgenden Ordnern und Dateien:

  • documentation - enthält ausführliche Anleitung zur Installation sowohl der Engine als auch der Vorlage;
    • documentation.html ist Hauptdokumentationsdatei. Sie lesen sie gerade :)
  • screenshots - enthält Bildschirmfotos vom Theme;
  • themeXXX - enthält:
    • themeXXX(full).zip - das Archiv mit der OpenCart Engine, Vorlage und Demodaten vom Theme (es wird für die Installation bei Null benutzt);
    • themeXXX(update).zip - das Archiv mit der Vorlage ohne Demodaten (es wird für die Installation auf bestehenden Online-Shop benutzt);
    • themeXXX(extensions).zip - das Archiv mit xml-Dateien;
  • info.txt - die Anleitungen zum Entpacken von Quelldateien;
  • sources_XXXXXXXX.zip - das Archiv mit Quelldateien. Es enthält:
    • psd - enthält die Quelldatei der Vorlage im Format Adobe Photoshop (.psd);
    • SCSS - enthält die CSS-Quelldateien, die mit Hilfe von SCSS geschrieben wurden.

Vorbereitung Was man vor allem machen soll

Bevor Sie anfangen, Ihre OpenCart Website zu installieren, müssen Sie alles gründlich vorbereiten.

Software

Damit die OpenCart Vorlage korrekt funktioniert, empfehlen wir Ihnen alle erforderlichen Anwendungen herunterzuladen. Die Liste der erforderlichen Software befindet sich auf der Vorschauseite Ihrer Vorlage. Anforderungen an verschiedene Vorlage können sich unterscheiden, deswegen nennen wir nur allgemeine Anforderungen:

  1. Sie benötigen das Programm Adobe Photoshop. Es wird bei der Bearbeitung von Quelldateien .PSD benutzt. Außerdem brauchen Sie dieses Programm, wenn Sie entweder das Grafikdesign oder Bilder Ihrer Vorlage ändern möchten.
  2. Um den Quellcode Ihrer Vorlagendateien zu bearbeiten, sollen Sie den Code-Editor wie, zum Beispiel, Adobe Dreamweaver, Notepad++, Sublime Text usw. nutzen.
  3. Um Dateien aufs Hosting hochzuladen, brauchen Sie den FTP-Manager wie Total Commander, FileZilla, CuteFTP usw.
  4. Nutzen Sie die folgenden Anleitungen, um Archive mit Hilfe von WinZip(PC) und StuffitExpander(MAC) zu entpacken.

Hosting

OpenCart basiert auf solche Anwendungen wie PHP/MySQL, aus diesem Grund müssen Sie den Hosting-Server vorbereiten.

Wenn Sie einen aktiveren Server haben, müssen Sie dann überprüfen, ob er den Systemanforderungen entspricht und ob Ihre OpenCart Website auf ihm korrekt funktionieren wird.

Sie können OpenCart auch auf Ihrem Computer mit Hilfe von einem lokalen Server installieren. Um einen lokalen Hosting-Server zu erstellen, müssen Sie die Software wie WAMP, AppServ, MAMP usw. nutzen. Jede von ihnen wird wie ein Programm, das OpenCart unterstützt, installiert.

Sie können die folgenden Anleitungen nutzen, um den lokalen Server einzustellen:

  • Wie man die Webentwicklungsumgebung wie AppServ installiert
  • Wie man die Webentwicklungsumgebung wie WAMP installiert
  • Wie man die Webentwicklungsumgebung wie XAMP installiert.

Installation auf bestehenden Online-Shop

Folgen Sie dieser Installationsart, wenn Sie schon eine OpenCart Website haben und ihr Aussehen ändern möchten, indem Sie ein neues Theme installieren.

 

Vollständige Installation

Die vollständige Installation wird benutzt, wenn Sie Ihre OpenCart Website bei Null installieren. Dafür müssen Sie alles, was oben geschrieben ist, vorbereiten und alle erforderlichen Programme herunterladen.

Erstellen der Datenbank

Nachdem Sie alle Dateien auf den Hosting-Server hochgeladen haben, müssen Sie eine neue Datenbank für Ihre OpenCart Website vorbereiten. Sie können sie mit Hilfe von dem Datenbank-Verwaltungswerkzeug im Hosting Control Panel (normalerweise PhpMyAdmin) erstellen.

Mit Hilfe von phpMyAdmin kann man die neue Datenbank in drei einfachen Schritten erstellen: (schauen Sie sich die folgenden Dias an):

  1. Drücken Sie auf den Button Databases (Datenbank) in dem oberen Navigationsmenü, um zur Liste der Datenbanken zu gelangen.
  2. Geben Sie den Namen der Datenbank ins Feld "Create new database (Neue Datenbank erstellen)" ein.
  3. Drücken Sie auf den Button rechts Create (Erstellen).

Die Datenbank wird in der Liste angezeigt. Klicken Sie auf sie, um den Zugriff auf die Datenbank zu erhalten:

 

  • 1. Öffnen Sie den Tab "Databases (Datenbank)", um zur Liste der Datenbanken zu gelangen.

  • 2. Geben Sie den Namen der Datenbank ein.
    3. Klicken Sie auf den Button "Erstellen".

  • 4. Die Datenbank wird in der Liste angezeigt. Klicken Sie auf sie, um den Zugriff auf die Datenbank zu erhalten.

 

Sie können auch die ausführliche Video-Anleitung Wie man eine Datenbank erstellt nutzen.

Wenn Sie Schwierigkeiten beim Erstellen der Datenbank haben, nehmen Sie den Kontakt mit dem Hosting-Anbieter auf.

Hochladen der Vorlage

Um mit OpenCart arbeiten zu können, müssen Sie die Dateien auf den Hosting-Server hochladen. Dies können Sie mit Hilfe von dem Dateimanager oder FTP-Manager machen.

  1. Öffnen Sie den Ordner Ihrer Vorlage.
  2. Finden Sie den Ordner 'themeXXX' und entpacken Sie das Archiv 'themeXXX(full).zip' auf Ihrem Computer. Den Inhalt des Archives laden Sie auf den Server ( Wie man die Dateien auf den Server hochlädt, Wie man die Dateien mit Hilfe vom cPanel (WebHost Manager, WHM) hochlädt) hoch.
  3. Legen Sie die richtigen Berechtigungen für die folgenden Ordner und Dateien der OpenCart Engine fest:
    • system/cache/;
    • system/logs/;
    • system/download/;
    • system/upload/;
    • image/;
    • image/cache/;
    • image/catalog/;
    • config.php;
    • admin/config.php.

    Die Berechtigungen müssen CHMOD 755 oder 777 abhängig von der Serverkonfiguration sein. Markieren Sie dafür die gewünschten Dateien und Ordner, wenn Sie mit dem Dateimanager arbeiten und finden Sie den Button "Berechtigungen ändern". Wenn Sie mit Linux arbeiten, brauchen Sie dann chmod -R 777 "Ordner_name".

Die Installation auf bestehenden Online-Shop

Überzeugen Sie sich davon, dass die Version Ihrer OpenCart Engine allen Anforderungen entspricht. Alle Versionen OpenCart sind auf der offiziellen Webseite OpenCart verfügbar.

  1. Um die OpenCart Vorlage zu installieren, müssen Sie ihre Dateien auf den Hosting-Server hochladen. Im Ordner Ihrer OpenCart Vorlage finden Sie das Archiv "themeXXX(update).zip" und entpacken Sie es auf Ihrem Computer.
  2. Nachdem laden Sie die entpackten Dateien in das Stammverzeichnis der installierten OpenCart Vorlage.
    Sie können auch die ausführliche Video-Anleitung nutzen Wie man die Dateien auf das Hosting hochlädt.
    Die Vorlagendateien schaden den Online-Shop in keinster Weise;
  3. Danach öffnen Sie Ihre Website im Browser und fügen Sie /install am Ende der Adresszeile hinzu (zum Beispiel, http://your_domain_name/opencart/install)
  4. Auf der Seite, die auf dem Bildschirm angezeigt ist, drücken Sie auf den Button Continue (Weiter). Aktualisierung vom Theme kann einige Zeit dauern.
  5. Loggen Sie sich ins Control Panel Ihrer Website ein, finden Sie Menü Erweiterungen -> Erweiterungen-Installateur (Extensions -> Extension Installer) und laden Sie die Dateien aus dem Archiv themeXXX(extensions) hoch. Nachdem Sie alle Archivdateien hochgeladen haben, finden Sie Menü Erweiterungen -> Modifikationen (Extensions -> Modifications) und drücken Sie auf den Button Aktualisieren(refresh) in der oberen rechten Ecke des Bildschirmes;
  6. Danach finden Sie die Einstellungen des Online-Shops System -> Einstellungen (System -> Settings), und im Tab Shop (Store) im Feld Vorlage (Template) wählen Sie das Theme - themeXXX aus.

Sie können auch die Anleitung zur Installation nutzen: OpenCart. Wie man die Vorlage auf bestehenden Online-Shop (mit Hilfe von dem Archiv “update”) installiert.

Die Einstellung von Erweiterungen

Die Installation von Modulen

Die Installation von Modulen wird in ein paar Schritten durchgeführt. Öffnen Sie das Admin Panel, indem Sie /admin zur Adresszeile www.yoursite.com/admin hinzufügen. Danach wählen Sie in der Seitenleiste den Tab Erweiterungen (Extensions) und in der DropDownListe Module (Modules) (Bild5) aus. In dem DropDownFenster sind alle installierten und fertigen zur Installation Module angezeigt. Wenn es neben dem Namen den Button Deinstallieren (Uninstall) gibt, bedeutet das, dass das Modul schon installiert wurde und wenn Sie auf den Button Bearbeiten (Edit) drücken, können Sie die Einstellungen dieses Moduls bearbeiten. Wenn Sie auf den Button Installieren (Install) drücken, installieren Sie dieses Modul. Wenn Sie das Modul entfernen möchten, sollen Sie dann auf den Button Deinstallieren (Uninstall) drücken, um das jeweilige Modul zu entfernen.

Achten Sie darauf, dass beim Entfernen des Moduls alle seinen Einstellungen auch entfernt werden, die es nicht möglich ist, wiederherzustellen.

Bild 5. Installation vom Modul

Nachdem Sie das Modul installiert haben, müssen Sie es auch einzustellen. Zwar können sich die Einstellungen abhängig von dem Modul unterscheiden, aber es gibt ein Pflichfeld Status (Bild 6) - das jeweilige Feld aktiviert/deaktiviert unser Modul.

Bild 6. Die Bearbeitung vom Modul

Um das Modul zur Seite hinzuzufügen, sollen Sie erst die Liste von Seiten finden. Es gibt zwei Möglichkeiten dafür (Bild 7):

  1. In der Seitenleiste finden Sie den Bereich System -> Design -> Layout (System -> Design -> Layout);
  2. Indem Sie sich in dem Modulbereich befinden, drücken Sie auf den Link here.

Bild 7. Die Auswahl von Seiten für die Bearbeitung

Wählen Sie eine Seite aus der Liste aus, zu der Sie das Modul hinzufügen möchten. Neben ihr drücken Sie auf den Button Bearbeiten (Edit) (Bild 8).

Bild 8. Die Auswahl einer Seite für die Bearbeitung

Hier finden Sie eine Liste der Module und ihrer Positionen, die schon auf der Seite sind (Bild 9).

Bild 9. Hinzufügen von Modulen zur Seite

Es sind die folgenden Positionen verfügbar (nach der Anzeigereihenfolge):

  1. Header Top (nach der Sektion Header);
  2. Column-left (linke Seitenleiste);
  3. Column-right (rechte Seitenleiste);
  4. Content-top;
  5. Content-bottom;
  6. Footer.

Das Feld Sortierreihenfolge (Sort Order) zeigt die Anzeigereihenfolge von Modulen auf der Seite, die sich in einer Positione befinden. Um das Modul zur Seite hinzuzufügen, drücken Sie auf den Button Modul hinzufügen (Add Module). In der Zeile, die Sie gerade sehen, wählen Sie das gewünschte Modul und seine Position auf der Seite aus, drücken Sie auf den Button Speichern (Save). Die Modulinstallation ist abgeschlossen.

Erstellen von Bannern

Modifikation TM Description for banners

Die Modifikation wird gebraucht, um die Beschreibungen von Bannern hinzuzufügen. Um die Modifikation zu entfernen oder zu deaktivieren, öffnen Sie Erweiterungen -> Modifikation (Extensions -> Modifications) finden Sie in der Liste "TM Description fo banners" Erweiterung und drücken Sie auf den Button Deaktivieren (Disable) oder Entfernen (Delete).

Die Einstellungen einiger Module haben das Feld zur Auswahl vom Banner, das gebraucht wird. Finden Sie den Bereich System -> Design -> Banner (System -> Design -> Banner) (Bild 10).

Bild 10. Die Seite von Bannern

Auf der jeweiligen Seite ist die Liste von erstellten Bannern angezeigt. Um ein neues Banner zu erstellen, drücken Sie auf den Button Neu hinzufügen (Add New), um das aktuelle Banner zu bearbeiten, drücken Sie auf den Button Bearbeiten (Edit).

Bild 11. Die Bearbeitung vom Banner

Geben Sie jetzt Banner Name, Status an und drücken Sie auf den Button Banner hinzufügen (Add Banner). In der Tabelle, die Sie gerade sehen, legen Sie das Folgende fest:

  1. Bannername;
  2. Beschreibung - der Text, der auf dem Banner angezeigt wird (html-Markup wird unterstützt);
  3. Link - legen Sie den relativen Link fest. Es ist der Link relativ zur Website-Adresse (zum Beispiel, index.php?route=product/product&product_id=47) die jeweilige Option ist nicht obligatorisch;
  4. Bild;
  5. Reihenfolge des Banners. Sie können mehr als ein Banner haben, deswegen muss dieses Feld unbedingt ausgefüllt werden.

Um das Banner zu entfernen, drücken Sie auf den Button Entfernen (Remove). Nachdem Sie das Banner erstellt haben, können Sie es für andere Module nutzen.

Parallax

Parallax Scrolling ist eine spezielle Scrolling-Technik in der Computergraphik. Dank ihm bewegen sich die Bilder im Hintergrund langsamer als Inhalt, der sich im Vordergrund befindet. Dadurch wird die Illusion von Tiefe und der Effekt des Eintauchens in ein zweidimensionales Videospiel geschaffen. Dank diesem Modul fügen Sie den Parallax-Effekt zur Website ohne zusätzliche Fähigkeiten und Anstrengungen hinzu.

Nachdem Sie dieses Modul installiert haben, können Sie es auf der Seite der Moduleinstellungen einstellen (Bild 12).

Bild 12. Die Moduleinstellung Parallax

Einstellungen:

  1. Module Name - Modulname;
  2. Banner - wählen Sie aus der Liste von Bannern, die Sie vorher erstellt haben, aus;
  3. Width - die Bildbreite für das Modul;
  4. Height - die Bildhöhe für das Modul;
  5. Status - der Modulstatus (aktiviert/deaktiviert).
Sowohl die Breite als auch die Höhe des Bildes müssen dem Originalbild, das standardmäßig benutzt wird, entsprechen. Andererseits wird der Effekt Ihren Erwartungen nicht entsprechen.

TM Google Map

Das jeweilige Modul ermöglicht es eine Google Karte mit einer unbegrenzten Anzahl von Markern zur Ihren Website (Bild 23) hinzuzufügen.

Beim Drücken auf den Marker, wird das Fenster mit der Adresse, die in demselben Tab im Feld Adresse (Address) festgelegt wurde, angezeigt.

Nachdem Sie das Modul installiert haben, können Sie es auf der Konfigurationsseite einstellen (Bilder 24-25).

Bild 24. Die Modulbearbeitung TM Google Map

Bild 25. Die Modulbearbeitung TM Google Map

Einstellungen:

  1. Module Name - der Modulname;
  2. Google Map API key - API-Schlüssel für Google Map. Die ausführliche Information ist auf der offiziellen Website verfügbar;
  3. Status - der Modulstatus (aktviert/deaktiviert);
  4. Map Type - Kartentyp. Es sind 4 Werte verfügbar:
    • Roadmap - die Straßenkarte, die standardmäßig benutzt wird ;
    • Terrain (Landscape) - topografischen Karte basierend auf Geländeinformationen;
    • Hybrid - eine Kombination aus normalen Karten und Satellitenbildern;
    • Satellite - Google Earth Bilder, die über den Satellit aufgenommen wurden;
  5. Map Zoom - Zahlenwert. Anfangsskalierung, die bei der Anzeige der Karte benutzt wird. Der Wert 0 entspricht dem niedrigsten Zoomniveau. Wenn Sie diesen Wert erhöhen, wird die Skalierung der Karte auch erhöht;
  6. Sensor - zeigt, ob es in der Anwendung der Sensor benutzt wird, der den Standort des Benutzers definiert.
  7. Map Width - die Kartenbreite. Sie müssen Maßeinheiten (% oder px) angeben;
  8. Map Height - die Kartenhöhe. Sie müssen Maßeinheiten (% oder px) angeben;
  9. Map Styles - legt den Kartenstil fest. Verschiedene Stile sind unter https://snazzymaps.com/ verfügbar;
  10. Disable UI - aktiviert/deaktiviert die Benutzerschnittstelle standardmäßig;
  11. Scrollwheel - aktiviert/deaktiviert die Änderungen von Kartenerweiterungen mit dem Mausrad;
  12. Draggable - aktiviert/deaktiviert die Möglichkeit der Kartenverschiebung;
  13. Marker Image - legt das Marker-Bild fest. Wenn Sie das Feld leer lassen, wird dann ein Standardmarker angezeigt;
  14. Active Marker Image - legt das Markerbild beim Drücken fest;
  15. Marker Image Width - legt die Breite des Markerbildes fest;
  16. Marker Image Height - legt die Höhe des Markerbildes fest.

Um den Marker hinzuzufügen, drücken Sie auf den Button Add New. Im Feld Geocode geben Sie die Markerkoordinaten (Breite, Länge - Beispiel:40.6700, -73.9400) und im Feld Address die Information, die beim Klicken auf den Marker angezeigt wird, an. Die Koordinaten eines gewünschten Standortes können Sie unter http://www.mapcoordinates.net finden. TM Cookie Policy

Das jeweilige Modul zeigt den Benutzern, ob es Cookies (cookies) (Bild 27) benutzt werden. Laut dem Gesetz über Cookies zeigt dieses Modul den Benutzern, ob Cookies verwendet werden.

Cookies sind kleine Textdateien, die Websites auf Computern der Benutzer erstellen. Normalerweise werden sie benutzt, um den Benutzer zu identifizieren. Außerdem werden sie benutzt, damit der Benutzer die Website bequem nutzen kann. Cookies sind ein fester Bestandteil mehrerer Funktionen auf der Website.

Zum Beispiel, mit Hilfe von Cookies werden diejenigen Produkte gespeichert, die Sie in dem Online-Shop zum Warenkorb hinzugefügt haben. Außerdem speichern sie Anmeldedaten oder stellen wertvolle Nutzungsstatistik und Informationen den Webseiteninhaber zur Verfügung.

Wenn Sie eine Website haben, werden Cookies höchstwahrscheinlich auf dieser Webseite benutzt. Ihre Hauptfunktion besteht darin, dass sie der Anzahl der Benutzer und ihre Interessen über spezielle Werkzeuge wie, zum Beispiel, Google Analytics folgen. Außerdem werden sie benutzt, um Anzeigen darzstellen. Wenn Sie einen Online-Shop schon haben, sind sie ein wesentlicher Bestandteil vom E-Commerce.

Am 26. Mai 2012 hat ICO Organisation einen Akt darüber veröffentlicht, dass "stillschweigende Zustimmung" nicht erlaubt wird. Aus diesem Grund müssen Sie unbedingt zeigen, wo auf der Website sowohl Cookies als auch Datenschutzrichtlinie sind.
